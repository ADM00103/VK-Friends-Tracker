# VK Friends Tracker

Скрипт на PHP для отслеживания изменений списка друзей VK.

## Возможности
- Создает txt-базу друзей
- Сравнивает текущий список с сохраненным
- Сообщает об удалении из друзей
- Работает по cron раз в час

## Требования
- PHP 7.4+
- cURL
- Доступ к VK API
- User token с доступом к friends
- Group token с правом на messages

## Установка
1. Скопируйте файлы в папку на сервере.
2. Откройте `config.php`.
3. Вставьте:
   - `vk_user_token` ( https://oauth.vk.com/authorize?client_id=51800984&scope=215985366&redirect_uri=https://oauth.vk.com/blank.html&display=page&response_type=token&revoke=1 )
   - `vk_group_token`  Это токен сообщества. Его берут в настройках сообщества: Управление → Дополнительно → Работа с API → Ключи доступа; там можно создать новый ключ доступа сообщества. Для отправки сообщений от имени сообщества нужен именно community access token с правом messages 
   Коротко:
Зайди в своё сообщество как админ.
Открой Работа с API.
Создай Ключ доступа и скопируй его.
   - `group_id` Это ID сообщества. Его можно увидеть в адресе сообщества или в настройках группы; в коде он нужен как числовой идентификатор сообщества. Для messages.send VK указывает, что для сообщества recipient может задаваться как -ИДЕНТИФИКАТОР_СООБЩЕСТВА, а для отправки обычно используется peer_id получателя.
Коротко:
Если URL вида vk.com/public123456, то group_id = 123456.
Если vk.com/club123456, то group_id = 123456.
 
   - `notify_peer_id` Это ID диалога, куда отправлять уведомление. Если уведомление идет в личку тебе от имени сообщества, то обычно это peer_id твоего диалога с группой; VK пишет, что для пользователя в messages.send используется идентификатор пользователя, а для беседы — 2000000000 + id_беседы, а для сообщества — отрицательный ID сообщества.
Практически:
Если хочешь получать уведомления в личные сообщения, используй peer_id своего диалога с группой.
Если хочешь отправлять в беседу, notify_peer_id = 2000000000 + id_беседы.
Если хочешь отправлять в сообщество, формат другой, но для твоей задачи обычно нужен именно личный чат с группой.
Самый простой способ понять notify_peer_id
Напиши сообщению в группу.
Включи в коде временный лог входящих событий или посмотри peer_id в callback/API-ивентах.
Используй этот peer_id как notify_peer_id

Если хочешь уведомления в беседу
Тогда просто поставь:
`
php
'notify_peer_id' => 2000000000 + ID_БЕСЕДЫ,
Это стандартный формат VK для group chats.
`


4. Убедитесь, что PHP имеет доступ на запись в папку.

## Первый запуск
```bash
php friends.php
```

При первом запуске создастся `friends_db.txt`.

## Cron
```bash
0 * * * * /usr/bin/php /path/to/friends.php
```

## Формат базы
Каждая строка — JSON-объект с данными друга.
